HTML

Pembelajaran awal untuk membuat sebuah web adalah belajar mengenai HTML. Berdasarkan code program di atas, maka dapat kita analisis strukturnya sebagai berikut.

  • Tag pada HTML umumnya memiliki pasangan yang memiliki nama tag sama. Perbedaanya ada pada tag terakhir di berikan tanda (/).
  • Tag (html) memberitahukan pada web browser bahwa merupakan awal dokumen HTML dan di akhiri dengan (/html).
  • Tag (head) dan (/head) merupakan teks informasi header. Informasi header ini tidak akan di tampilkan pada jendela browser.
  • Tag (title) dan (/title) di gunakan untuk judul dokumen. Teks pada tag ini akan di tampilkan pada title bar di web browser.
  • Pada tag (body) dan (/body) merupakan teks yang akan di tampilkan pada web browser.
